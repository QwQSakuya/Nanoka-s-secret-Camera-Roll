
# 奈叶香的秘密摄影集
[English](README.md) | [简体中文](README_zh.md)

此项目参考和灵感来源 :
[Anan-s-Sketchbook-Check-Box](https://github.com/MarkCup-Official/Anan-s-Sketchbook-Chat-Box)

- 显示当前表情的悬浮窗
