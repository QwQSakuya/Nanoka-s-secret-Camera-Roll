
# 奈叶香的秘密摄影集
[English](README.md) | [简体中文](README_zh.md)

根据你输入的文字生成图片 和 快速发送导入的表情包

<small>此项目参考和灵感来源 :
[Anan-s-Sketchbook-Check-Box](https://github.com/MarkCup-Official/Anan-s-Sketchbook-Chat-Box)</small>
## 特殊功能(表情生成)
- 内置生成图片的编辑器
- 显示当前表情的悬浮窗
- 可设置特殊转换颜色的条件
## 特殊功能(快捷表情)
- 可导航的弹出窗口用于查看发送什么表情
- tag筛选, 快速搜索
- 支持 (png,gif,mp3,mp4) 
## 额额额额....
- 此程序完全基于bugs (咳咳咳)
- 还有很多未知bug
- 没有做任何的优化
- 98% ai 生成代码
- 请!!!!不要对此项目有高预期
- 如果你突然遇到bug,可以先查看'common fix bug'再report
