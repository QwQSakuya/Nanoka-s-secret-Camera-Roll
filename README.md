
# Nanoka's secret Camera Roll
[English](README.md) | [简体中文](README_zh.md)


A tool that generates images base on your chatbox and adds a shortcut for sending emotes

<small>the generating part is inspired by :  
- [Anan-s-Sketchbook-Check-Box](https://github.com/MarkCup-Official/Anan-s-Sketchbook-Chat-Box)</small>
## Special functions (emote generating)
- Generates images with a shortcut
- Built in editor for the images
- A floating gui to show the current emote
- Special color changing symbol
## Special functions (custom import emotes)
- Navagational popup gui for sending stored emotes
- Fast search using tags
- supports (png,gif,mp3,mp4)

## How to use (special function)

After opening , setup the whitelist in settings

Here is some useful commands and default shortcuts (in a whitelisted program)

| Command | Description |
| :--- | :--- |
| `ctrl+a' | checks if the text entered is correct , if true summons a popup window to pick emots |
|`/*`| show all emots when selected by ctrl+a|
|`/*tag=...`| show all emots that have the entered tag after being selected by ctrl+a |
|`/{filename}`| show all emots in that is in the file after being selected by ctrl+a |
|`/{filename}/tag=...`| show all emots that are in the file with the entered tag after being selected by ctrl+a |
| :--- | :--- |
| `ctrl+enter` | turn the entered text into a photo |
